# FitLife Android App

A comprehensive workout planning Android application built with Java and Material Design 3.

## Features
- User authentication (registration & login)
- Create and manage workout routines
- Add exercises with sets, reps, and equipment
- Advanced gesture controls (swipe, shake, long-press, double-tap)
- SMS delegation to share workouts with contacts
- Material Design 3 UI with purple and mint green color scheme

## Technical Stack
- **Language**: Java
- **Min SDK**: API 24 (Android 7.0)
- **Target SDK**: API 34 (Android 14)
- **Database**: Room (SQLite)
- **UI**: Material Design 3
- **Image Loading**: Glide
- **Architecture**: MVC with Repository Pattern
